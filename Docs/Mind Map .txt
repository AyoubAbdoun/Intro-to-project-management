QATAR UNIVERSITY SOLAR TRAM PROJECT

│   ├─ Project Vision
│   │   ├─ Develop sustainable, autonomous solar-powered trams
│   │   ├─ Construct trams from recyclable metal containers
│   │   └─ Promote eco-friendly smart transportation in Qatar University
│   ├─ Purpose
│   │   ├─ Replace campus buses for internal transportation
│   │   ├─ Connect university buildings directly to metro stations
│   │   └─ Reduce traffic congestion and emissions on campus

│   ├─ Recyclable Construction
│   │   ├─ Repurpose metal containers into lightweight tram bodies
│   │   ├─ Modular frame for easy scalability and maintenance
│   │   └─ Incorporate corrosion-resistant materials for durability
│   ├─ Autonomous Driving System
│   │   ├─ AI navigation using pre-programmed routes and maps
│   │   ├─ Sensors: LIDAR, radar, ultrasonic for obstacle detection
│   │   └─ Remote monitoring and fail-safe manual override controls
│   └─ Mobile App & GPS
│       ├─ Real-time tracking and dynamic scheduling
│       ├─ Adjustable pickup radius (50–120 m) using geofencing
│       └─ BLE/Wi-Fi fallback and two-step confirmation for safety
