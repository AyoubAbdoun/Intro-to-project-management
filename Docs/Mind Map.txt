QATAR UNIVERSITY SOLAR TRAM PROJECT
│
├── Core Idea
│   ├─ Sustainable, autonomous solar-powered trams made from recyclable materials
│   ├─ Replaces campus buses and connects university buildings to the metro station
│   ├─ Students can stop trams via GPS-based mobile app within 50–100 m radius
│   └─ Designed to be Qatar’s first AI-driven smart transport initiative
│
├── Technology & Design
│   ├─ Solar Power Integration
│   │   ├─ Solar panels on tram roof and charging stations around campus
│   │   ├─ Battery Energy Storage System (BESS) for low sunlight hours
│   │   └─ Smart power management and grid-hybrid capability
│   ├─ Recyclable Construction
│   │   ├─ Reuse metal containers for tram carriages
│   │   └─ Lightweight, modular design for scalability and maintenance
│   ├─ Autonomous Driving System
│   │   ├─ AI-based navigation and obstacle detection
│   │   ├─ Sensors: LIDAR, radar, ultrasonic arrays
│   │   ├─ Central control AI for path adjustment and safe deceleration
│   │   └─ Remote override and monitoring system
│   └─ Mobile App & GPS
│       ├─ Real-time location tracking and tram scheduling
│       ├─ Dynamic pickup radius (50–120 m) with geofencing
│       └─ Two-step stop confirmation and BLE/Wi-Fi fallback for poor GPS
│
├── Infrastructure & Engineering
│   ├─ Civil Engineering Works
│   │   ├─ Rail/guideway path and stop placement across campus
│   │   └─ Pedestrian-safe tram lanes and shaded waiting areas
│   ├─ Electrical Engineering
│   │   ├─ PV integration with inverters and charge controllers
│   │   ├─ Electrical distribution and load management
│   │   └─ Maintenance of charging systems and energy flow
│   └─ Testing & Safety
│       ├─ Emergency braking and obstacle detection validation
│       ├─ Passenger safety evacuation testing
│       └─ Fail-safe redundancy and manual control integration
│
